# Git Repository → Presentation Generator MVP 기술명세서

## 프로젝트명
- 국문: Git 저장소 기반 자동 프레젠테이션 생성기
- 영문: Git Repository Presentation Generator

---

# 1. 프로젝트 개요

## 목표
GitHub 저장소 URL 하나만 입력하면:

1. 저장소 전체 소스코드 분석
2. Reverse Engineering 기반 기술명세서 생성
3. 기술명세서 + 소스코드 기반 발표용 슬라이드 생성
4. README / 기술명세서 / PPT / PDF 다운로드 제공

까지 자동으로 수행하는 AI 기반 웹앱 MVP를 구현한다.

초기 MVP는 “개발 프로젝트를 빠르게 이해하고 발표 자료까지 자동 생성”하는 흐름에 집중한다.

# 25. 결론

이 프로젝트의 핵심은 단순한 문서 생성이 아니라:

```text
소스코드 → Reverse Engineering → 기술명세서 → 발표자료
```

로 이어지는 “AI 기반 자동 프로젝트 해석 시스템” 구축이다.

즉, 개발자가 프로젝트를 설명하지 않아도:

- 코드 구조를 이해하고
- 아키텍처를 분석하며
- 발표 가능한 수준의 문서를 생성하는

AI Engineering Workflow 자동화 플랫폼을 목표로 한다.
